# review_books

## サイトの概要
### サイトテーマ

本の好きな人が本のレビューやコメントや評価などを行い、評価が高い順でランキングの行われていてジャンル検索もあるサイト

### テーマを選んだ理由
現在の自分の実力で何が作れるかを考えた時にbookersの延長線上のものかと思い、それに加えて本を読むのが好きなのでそのレビューをするサイトにしようと決めました。

### ターゲーットユーザー

本の好きな人
### 主な利用シーン
本の好きな人が本に対してレビューや評価、コメントなどを書き、その中でもっとも評価が高い順にランキングする
ジャンル検索もできる
